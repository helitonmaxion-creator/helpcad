# HELPCAD — Sistema de Monitoramento de Estoque

Dashboard de monitoramento de capacidade de estoque com velocímetros em tempo real.

## 🗂 Estrutura do Repositório

```
helpcad/
├── index.html      ← App principal (GitHub Pages / público)
├── .gitignore      ← Protege o config.js
└── README.md       ← Este arquivo
```

## 🔒 Arquivo local (NÃO está no GitHub)

```
📁 Pasta no seu PC:
├── loader.html     ← Abre este no navegador
└── config.js       ← Credenciais (NUNCA enviar ao GitHub)
```

## 🚀 Como usar

1. Abra sempre o arquivo **`loader.html`** no Chrome
2. Ele carrega automaticamente a versão mais recente do GitHub
3. Edite o `index.html` no repositório → na próxima abertura já aparece atualizado

## ⚙️ Tecnologias

- **Frontend**: HTML/CSS/JS puro — sem frameworks
- **Banco de dados**: [Supabase](https://supabase.com)
- **Gráficos**: Chart.js
- **Fonte**: Google Fonts (Orbitron, Rajdhani)
